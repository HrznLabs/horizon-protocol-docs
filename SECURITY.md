| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of our software seriously. If you find any security vulnerability in the Horizon Protocol documentation or related repositories, please report it to us immediately.

### How to Report

Please **DO NOT** create a public GitHub issue. Instead, report the vulnerability via email:

- **Email**: security@horizonprotocol.xyz

Please include as much information as possible to help us reproduce and fix the issue. We will acknowledge your report within 48 hours.

### Disclosure Policy

- We ask that you do not disclose the vulnerability publicly until we have had the chance to fix it.
- We will work with you to resolve the issue promptly.
