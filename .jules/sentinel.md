## 2024-10-24 - Transitive Dependency RCE Vulnerabilities
**Vulnerability:** A high severity RCE vulnerability was found in `serialize-javascript` via `RegExp.flags` and `Date.prototype.toISOString()`, introduced as a transitive dependency deep within the `@docusaurus/preset-classic` tree.
**Learning:** In a heavily abstracted project using complex presets (like Docusaurus), deeply nested transitive dependencies can harbor severe vulnerabilities that cannot be patched via simple `npm update` or bumping top-level dependencies. The standard practice for this repository is to mitigate these via the `resolutions` block in `package.json` to force a specific patch version across all transitive dependencies, but new vulnerabilities might be missed.
**Prevention:** Regularly run `yarn audit` and identify vulnerable packages. When a vulnerable transitive dependency is found, determine the secure version and explicitly add it to the `resolutions` block in `package.json`, rather than waiting for top-level framework updates.
